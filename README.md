## Overview
This repository provides an end-to-end analytical pipeline for studying remittance inflows to the Philippines using the Kaggle dataset:
https://www.kaggle.com/datasets/joshbuttler/remittance-to-the-philippines

The project covers:
- Data cleaning and preprocessing
- Exploratory and comparative analysis
- Statistical and econometric testing
- Forecasting and predictive modeling
- Segmentation and clustering

## Objectives
- Understand long-term remittance trends
- Identify geographic and channel-based differences
- Quantify statistical relationships
- Forecast future remittance flows
- Segment remittance behavior patterns

## Technologies
- Python
- pandas, numpy
- matplotlib, seaborn
- scipy, statsmodels
- scikit-learn
- prophet (optional)

## Dataset Source
Original dataset is sourced from Kaggle and must be downloaded manually:
https://www.kaggle.com/datasets/joshbuttler/remittance-to-the-philippines
